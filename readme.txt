This is a wordpress plugin that removes or hide date of a particular post or page in your wordpress blog.

== Description ==
This is a plugin that I made back when I’m working for a specific internet marketer. He was actually looking for ways on how to get rid of the post dates on his hundreds of niche marketing sites. So, this is the plugin that I made to solve the problem.

This is a wordpress plugin that removes or hide date of a particular post or page in your wordpress blog. If you have a problem just like what I’ve mentioned above or just want to really remove the date of your content in your wordpress sites, then this plugin is the solution for you.

= Features =

*   These are the areas that this plugin will remove your dates: homepage, post, page
*   No option page. It just works! ;)
*   Works almost on all of the existing Wordpress themes out there, including Thesis Template.
*   Works fine on most of WordPress Themes, including 3.0+ WordPress version and Thesis template.

== Installation ==

*   Download the plugin.
*   Upload it to your WordPress Plugin Dashboard
*   Activate the plugin

And that’s it. If you want to deactivate the plugin, just go back to your WordPress Plugin Dashboard, find this plugin and click deactivate.
